To run the example project, clone the repo, and run pod install from the Example directory first.

Requirements

Installation

SuperDDAIuPay is available through CocoaPods. To install it, simply add the following line to your Podfile:

pod 'SuperDDAIuPay'

Add the following line inside didFinishLaunchingWithOptions on app delegate:

UIFont.loadIuPayFonts
